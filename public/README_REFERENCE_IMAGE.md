# 참조 이미지 안내

## 필요한 파일

이 폴더에 `gd-reference.jpg` 파일을 추가해주세요.

### 요구사항:
- 파일명: `gd-reference.jpg`
- 포맷: JPG/JPEG
- 권장 해상도: 1024x1024 이상
- 내용: 지드래곤(G-DRAGON)의 얼굴이 명확하게 보이는 사진

### 이미지 선택 가이드:
- 정면 또는 약간 측면 각도
- 조명이 좋고 얼굴이 선명한 사진
- 이목구비가 잘 보이는 고화질 이미지
- 얼굴에 가려지는 요소가 최소화된 사진

## 사용 목적

이 참조 이미지는 AI가 GD의 얼굴 특징을 정확하게 파악하여 초상화를 생성할 때 사용됩니다.

**중요**: 참조 이미지가 없으면 `/art` 기능이 작동하지 않습니다.
